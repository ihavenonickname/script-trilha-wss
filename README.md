# Script trilha WSS

## Objetivo

Automatizar processos de recon passivo e ativo utilizando ferramentas nativas do Kali Linux como building blocks de um script bash.

- Enumeração de subdomínios
- Probe de subdomínios
- Enumeração de diretórios
